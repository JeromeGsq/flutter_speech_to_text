# flutter_speech_to_text Example

A demo app showcasing the `flutter_speech_to_text` Flutter plugin for real-time speech-to-text conversion.

## Getting Started

1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`

### iOS Setup

Make sure to run `pod install` in the `ios/` directory:

```bash
cd ios && pod install
```

### Android Setup

No additional setup required. The `RECORD_AUDIO` permission is already configured.

## Features Demonstrated

- Real-time speech recognition
- Partial and final transcription results
- Confidence scores
- Permission handling
- Error handling
